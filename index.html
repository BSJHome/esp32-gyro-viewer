WiFi

Hvis ESP32 stadig står på et gammelt netværk, så tryk først Ryd alle WiFi refs, forbind igen via Serial/ny upload hvis nødvendigt, og gem derefter det rigtige WiFi. I v6 er WiFi låst til Arduino-koden ved boot. HTML kan ikke ændre SSID/password permanent. Ret DEFAULT_WIFI_1_SSID og DEFAULT_WIFI_1_PASS øverst i Arduino-koden, upload igen, og brug derefter HTML kun til visning/filter/hastigheder.
WiFi aktiv
OFF = slukker kun midlertidigt. Efter reboot tænder WiFi igen som rescue.

Sampling / hastigheder

5 ms ≈ 200 Hz. Lavere tal = hurtigere sampling.
Dette er hvor ofte ESP32 sender data til browseren.
0 = opdatér 3D ved hver MQTT-pakke. 50 ms ≈ 20 FPS.
0 = log hver MQTT-pakke.
